# プライバシーポリシー

_最終更新日: 2025年1月_

## 概要

QuickTabby はお客様のプライバシーを保護することをお約束します。このポリシーでは、収集するデータとその使用方法について説明します。

## 単一目的

QuickTabby には明確な単一の目的があります：**最近使った順にソートされた高速タブ切り替え**

すべての機能はこのコア機能をサポートしています：

- 最近の使用順（MRU）でソートされたタブリスト
- 素早いアクセスのためのキーボードファーストナビゲーション
- 視覚的な識別のためのタブサムネイル
- 包括的なタブ管理のためのマルチウィンドウ対応

この拡張機能は、関連のない機能をバンドルしていません。

## データ収集

**QuickTabby は、外部サーバーに個人データを収集、送信、保存することはありません。**

すべてのデータは Chrome の組み込みストレージ API を使用して、お客様のデバイスにローカルで保存されます。

## ローカルに保存されるデータ

QuickTabby は以下のデータをデバイスにローカルで保存します：

### タブ使用データ

- **最近使った順**: 最近の使用順にソートされたタブ ID のリスト
- **タブサムネイル**: 視覚的なプレビュー用のタブコンテンツのキャッシュされたスクリーンショット

このデータは以下を使用して保存されます：

- `chrome.storage.local`: タブ順序データと設定
- IndexedDB: タブサムネイル

### スクリーンショットに関する重要なお知らせ

**警告**: タブのスクリーンショットには、キャプチャ時に画面に表示されていた機密情報が含まれる可能性があります。これには以下が含まれます：

- 個人情報または機密情報
- パスワードや認証情報
- プライベートな通信内容
- 財務情報

**QuickTabby はこのデータを保護しません**。お客様のデバイスやブラウザにアクセスできる他のユーザーから保護されることはありません。コンピュータにアクセスできる人は誰でも、保存されたスクリーンショットを閲覧できる可能性があります。

データの責任に関する完全な規約については、[利用規約](/ja/terms)をご覧ください。

### ユーザー設定

- 表示設定
- キーボードショートカット設定
- ウィンドウモード設定

## 使用する権限

QuickTabby は以下の Chrome 権限を必要とします：

| 権限        | 用途                                                                    |
| ----------- | ----------------------------------------------------------------------- |
| `tabs`      | 表示と切り替えのためのタブ情報（タイトル、URL、ファビコン）へのアクセス |
| `storage`   | タブ順序データとユーザー設定をローカルに保存                            |
| `activeTab` | 現在アクティブなタブのサムネイルをキャプチャ                            |
| `alarms`    | 期限切れのサムネイルデータのクリーンアップをスケジュール                |
| `offscreen` | システムカラースキームの検出                                            |

## サードパーティサービス

QuickTabby はサードパーティの分析、トラッキング、広告サービスを一切使用しません。

## データ共有

サードパーティとデータを共有することはありません。すべてのデータはお客様のデバイスに留まります。

## データ削除

QuickTabby のすべてのデータを削除するには：

1. QuickTabby アイコンを右クリック
2. 「Chrome から削除」を選択

これにより、タブ履歴やサムネイルを含む、ローカルに保存されたすべてのデータが削除されます。

## お客様の責任

QuickTabby はすべてのデータをローカルに保存し、外部に送信することはありませんが、以下についてはお客様の責任となります：

- 不正アクセスからデバイスを保護すること
- ブラウザプロファイルへのアクセス権限を持つ者を管理すること
- ローカルに保存されたデータはデバイスにアクセスできる人なら誰でもアクセス可能であることを理解すること

詳細については[利用規約](/ja/terms)をご覧ください。
