# キーボードショートカット

## QuickTabby を開く

| ショートカット | アクション |
|----------------|------------|
| `Alt+Q` | ポップアップを開く |
| `Alt+Shift+P` | ポップアップを開く（サブ） |

各ポップアップコマンドには独自のモード設定（全ウィンドウまたは現在のウィンドウ）があり、オプションで設定できます。

### 再押下で選択

「ポップアップを開く（サブ）」ショートカットは「再押下で選択」機能をサポートしています。有効にすると、ポップアップが開いている状態でショートカットを再度押すと、選択中のタブに切り替えてポップアップを閉じます。

:::warning
Chrome 拡張機能の制限により、「再押下で選択」は**「ポップアップを開く（サブ）」でのみ利用可能**です。メインの「ポップアップを開く」ショートカット（`Alt+Q`）は、Chrome の処理方法が異なるため、この機能をサポートしていません。

この機能を使用したい場合は、`chrome://extensions/shortcuts` で「ポップアップを開く（サブ）」にお好みのショートカットを設定してください。
:::

## ポップアップ内ナビゲーション

| ショートカット | アクション |
|----------------|------------|
| `↑` / `k` | 選択を上に移動 |
| `↓` / `j` | 選択を下に移動 |
| `Enter` | 選択したタブに切り替え |
| `Esc` | ポップアップを閉じる |

## 検索

| ショートカット | アクション |
|----------------|------------|
| 入力開始 | タイトルまたは URL でタブをフィルタリング |
| `Backspace` | 検索文字を1文字ずつ削除 |
| `Esc` | 検索をクリア（検索がアクティブな場合） |

## タブナビゲーション

| ショートカット | アクション |
|----------------|------------|
| `Alt+Shift+H` | 左のタブに切り替え（端でループ） |
| `Alt+Shift+L` | 右のタブに切り替え（端でループ） |

## カスタマイズ

Chrome でキーボードショートカットをカスタマイズできます：

1. `chrome://extensions/shortcuts` にアクセス
2. QuickTabby を見つける
3. お好みのショートカットを設定

:::tip
上記のショートカットはデフォルトです。お好みの組み合わせに変更できます。
:::
