# 機能

QuickTabby は、タブの切り替えを高速かつ効率的にする Chrome 拡張機能です。

## コア機能

### 最近使った順

QuickTabby は最近使用したタブを自動的に追跡します。ポップアップを開くと、タブは使用順にソートされ、関連性の高いタブが常に上部に表示されます。

### キーボードファーストナビゲーション

マウスを使わずにタブを操作できます：

- **Alt+Q** - QuickTabby ポップアップを開く
- **↑/↓** または **j/k** - 選択を上下に移動
- **Enter** - 選択したタブに切り替え
- **Esc** - ポップアップを閉じる

### マルチウィンドウ対応

すべてのブラウザウィンドウのタブを切り替えるか、現在のウィンドウに絞り込むことができます：

- **Alt+Shift+A** - 全ウィンドウモード
- **Alt+Shift+C** - 現在のウィンドウモード

### タブサムネイル

タブの視覚的なプレビューを表示して、目的のタブをすばやく見つけられます。

## カスタマイズ機能

### テーマ設定

3つのテーマモードから選択できます：

| モード | 説明 |
|--------|------|
| **ライト** | 常にライトテーマを使用 |
| **ダーク** | 常にダークテーマを使用 |
| **自動** | OSのテーマ設定に自動的に合わせる |

自動モードはOSの設定を検出し、昼夜を問わずシームレスな体験を提供します。

### プライバシー設定

機密性の高いコンテンツのスクリーンショット処理を制御できます：

#### スキップパターン

スクリーンショットをキャプチャしないURLパターンを定義します。以下のような場合に便利です：

- 銀行サイト
- パスワードマネージャー
- 社内ツール

デフォルトのスキップパターン：

- `chrome://*`
- `chrome-extension://*`
- `edge://*`
- `about:*`

#### ぼかしパターン

スクリーンショットをキャプチャしつつ、ぼかし効果で表示するURLパターンを定義します。プライバシーを保護しながら、タブの視覚的なコンテキストを表示できます。

**パターン形式**: `*://example.com/*` や `*://*.bank.com/*` のようなマッチパターンを使用

### 複数キーバインド

同じアクションに複数のキーボードショートカットを割り当てられます：

- 各アクションに複数のキーバインドを設定可能
- オプションページでショートカットをカスタマイズ
- 修飾キーをサポート：Ctrl、Alt、Shift、Cmd/Meta

例えば、選択を下に移動するアクションに `j` と `↓` の両方を設定でき、Vimスタイルと従来のナビゲーションの両方に対応できます。

カスタマイズ可能なショートカットの全リストは[キーボードショートカット](/ja/shortcuts)をご覧ください。
