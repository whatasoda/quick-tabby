# 機能

QuickTabby は、タブの切り替えを高速かつ効率的にする Chrome 拡張機能です。

## コア機能

### 最近使った順

QuickTabby は最近使用したタブを自動的に追跡します。ポップアップを開くと、タブは使用順にソートされ、関連性の高いタブが常に上部に表示されます。

### キーボードファーストナビゲーション

マウスを使わずにタブを操作できます：

- **Alt+Q** - QuickTabby ポップアップを開く
- **↑/↓** または **j/k** - 選択を上下に移動
- **Enter** - 選択したタブに切り替え
- **Esc** - ポップアップを閉じる

### あいまい検索

タイトルやURLの一部を入力してタブをすばやく検索できます：

- **あいまいマッチング** - タイプミスや部分一致でもタブを見つけられます
- **スマートランキング** - 検索結果はマッチ度順にソートされ、最も一致するものが先頭に表示されます
- **マッチハイライト** - タイトル/URLのどの部分がクエリにマッチしたか視覚的に確認できます
- **日本語対応** - ローマ字でひらがな/カタカナのタイトルを検索できます（例：「nihon」で「にほん」や「ニホン」を検索）

検索バーはオプションでショートカットコマンドごとに設定できます：

| モード | 動作 |
|--------|------|
| **常に表示** | 検索バーは常に表示され、フォーカスされます |
| **入力時に表示** | 入力を開始すると検索バーが表示されます |

### マルチウィンドウ対応

すべてのブラウザウィンドウのタブを切り替えるか、現在のウィンドウに絞り込むことができます。各ポップアップコマンド（`Alt+Q` と `Alt+Shift+P`）は、オプションで独自のモード設定ができます。

### タブナビゲーション

ポップアップを開かずに隣接するタブにすばやく切り替えられます：

- **Alt+Shift+H** - 左のタブに切り替え
- **Alt+Shift+L** - 右のタブに切り替え

端でループするため、最初のタブで左を押すと最後のタブに切り替わります。

### タブサムネイル

タブの視覚的なプレビューを表示して、目的のタブをすばやく見つけられます。

## カスタマイズ機能

### テーマ設定

3つのテーマモードから選択できます：

| モード | 説明 |
|--------|------|
| **ライト** | 常にライトテーマを使用 |
| **ダーク** | 常にダークテーマを使用 |
| **自動** | OSのテーマ設定に自動的に合わせる |

自動モードはOSの設定を検出し、昼夜を問わずシームレスな体験を提供します。

### プライバシー設定

機密性の高いコンテンツのスクリーンショット処理を制御できます：

#### スキップパターン

スクリーンショットをキャプチャしないURLパターンを定義します。以下のような場合に便利です：

- 銀行サイト
- パスワードマネージャー
- 社内ツール

デフォルトのスキップパターン：

- `chrome://*`
- `chrome-extension://*`
- `edge://*`
- `about:*`

#### ぼかしパターン

スクリーンショットをキャプチャしつつ、ぼかし効果で表示するURLパターンを定義します。プライバシーを保護しながら、タブの視覚的なコンテキストを表示できます。

**パターン形式**: `*://example.com/*` や `*://*.bank.com/*` のようなマッチパターンを使用

### 複数キーバインド

同じアクションに複数のキーボードショートカットを割り当てられます：

- 各アクションに複数のキーバインドを設定可能
- オプションページでショートカットをカスタマイズ
- 修飾キーをサポート：Ctrl、Alt、Shift、Cmd/Meta

例えば、選択を下に移動するアクションに `j` と `↓` の両方を設定でき、Vimスタイルと従来のナビゲーションの両方に対応できます。

カスタマイズ可能なショートカットの全リストは[キーボードショートカット](/ja/shortcuts)をご覧ください。
